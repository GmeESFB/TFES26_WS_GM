Four Learning Pathways Reference

How the differentiated framework structures content for every learner
🟩 Discovery Pathway
Bloom's: Remember / Understand

Adapted support for students with learning differences, language barriers, or low prior knowledge. Maximises scaffolding while maintaining access to the same core content.

Key features: Simplified vocabulary · Visual scaffolds & diagrams · Sentence starters · Fill-in-the-blank exercises · Worked examples · Bilingual glossary · Chunked activities (5-7 min)
🔵 Standard Pathway
Bloom's: Apply / Analyse

Meets curriculum-level expectations for the majority of students. Balanced support and autonomy. This is the baseline against which the other pathways are differentiated.

Key features: Grade-appropriate vocabulary · Application exercises · Structured analysis tasks · Some guided practice then independent work · Self-assessment checklists
🟣 Expert Pathway
Bloom's: Analyse / Evaluate

Advanced challenges for high achievers who need intellectual stimulation beyond the standard curriculum. Emphasises critical thinking, cross-disciplinary connections, and open-ended problem solving.

Key features: Advanced vocabulary · Open-ended problems · Research connections · Cross-disciplinary links · Minimal scaffolding · Peer teaching opportunities
🟠 Autonomy Pathway S6/S7
Bloom's: Evaluate / Create

Self-directed pathway for S6-S7 students preparing for the European Baccalaureate. Minimises teacher instruction, maximises student agency, planning, and metacognition. Students define their own learning objectives within the topic framework.

Key features: Learning contract · Self-study guide with curated resources · Investigation/project brief · Self-evaluation rubric · Bac-style practice questions · Timeline planning · Metacognitive reflection journal
