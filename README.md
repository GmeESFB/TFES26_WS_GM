## Deploy to GitHub Pages

1. Create a new **public** repository on GitHub (e.g. `tfes2026`).
2. Upload the entire contents of this folder to the repository root — or drag-and-drop the folder in the GitHub web interface.
3. Go to **Settings → Pages**.
4. Under *Source*, select **Deploy from a branch** → branch `main` → folder `/ (root)`.
5. Click **Save**. GitHub will publish the site in ~1 minute.
6. Your site will be live at: `https://<your-username>.github.io/tfes2026/`

> The `.nojekyll` file is already included — it tells GitHub Pages to serve the files as-is without Jekyll processing.
